Some things in the game can increase talent levels beyond lv10. This includes constellations, Childe's passive, some event effects. Different characters benefit from boosting different talent levels beyond lv10. Some benefit from boosting all, while some only need to boost one or two.

If a character can only pick one or two talents to boost beyond lv10, then there are talents that have certain priorities.
