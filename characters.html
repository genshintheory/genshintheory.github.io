General overview of gameplay mechanics of characters: this is like the path system of Star Rail or the role/job system of some other games like FF. As in other games, some characters can fulfill multiple roles.

Support

Sometimes the best defense is offense, and the best offense is defense.

Buffers/Offensive supports:

  • ATK Buffer: Bennett, Sara, any catalyst user with TTDS, Yoimiya (after using Q), certain constellation effects, etc.
  • Additive DMG buffer: Yun Jin, Shenhe
  • DMG Bonus buffer: Mona, Kazuha
  • DEF buffer: Gorou
  • Talent scaling buffer: Childe (passive increase NA level +1)
  • CRIT buffer: Faruzan C6, Rosaria, Sara C6, Gorou C6, Shenhe C2
  • Reaction buffer/enabler: Xingqiu (hydro applier, vaporize), Childe (hydro applier, vaporize)
  • EM buffer: Nahida (Q), Sucrose (EM), Kazuha (C2), instructor set wearers
  • RES shred buffer: any Anemo VV unit (Venti, Jean, Sucrose, Kazuha), Faruzan, Morax, superconduct units, Shenhe (Q), Eula (E), any unit via deepwood
  • DEF shred buffer: Raiden (C2), Yae (C6), Ayaka (C4), Lisa (A4), Klee (C2), Razor (C4)
  • Crowd Control (Vacuum): Venti, Faruzan, Sucrose, Kazuha, Jean, Anemo MC (AMC)
  • Battery: Raiden, Electro MC (EMC), Cyno (C4), usually bring corresponding element
  • Other (infusion): Chongyun (cryo infusion for shield shred), C6 Bennett, Candace

Defensive supports:

  • Healer: Jean, Qiqi, Barbara, Kokomi, Yaoyao, Diona, Bennett, Noelle, Gorou C4 (not much), Morax C6 (not much)
  • DMG reducer: Dehya, Jean (C6), Xingqiu, Beidou
  • Shield: Diona, Xinyan, Thoma, Beidou (C1), Yanfei (C4), Morax, Noelle, any Geo character via crystallize

Off field dps

DoT should last for more than a few seconds (else classify as nuke)

  • Through E: Dehya, Yae, Albedo, Fischl, Raiden E, Morax E (lesser extent)
  • Through Q: Beidou, Xiangling, Xingqiu, Kazuha, Venti, Yoimiya, Ganyu, Ayato
  • Through ocean clam bubble pop: Qiqi

Term sub-dps sometimes refers to off field dps or burst dps.

  • Mainly CA: Itto, Ganyu, Hu Tao, Tigh, Ayaka, Lyney
  • CA/NA: Itto, Childe, Scaramouche, Raiden
  • Mainly NA: Yoimiya, Cyno, Diluc, Eula, Razor, Kokomi

Uptime

  • 100% uptime (or near 100% uptime): Xiao, Diluc, Cyno, Eula, Alhaitham, Ganyu, Ayaka, Itto (C2+), Childe (C6), Lyney
  • Downtime: Raiden (11s), Itto (C0, C1), Childe, Hu Tao, Yoimiya, Ayato (6s), Scaramouche (6s), Tigh
