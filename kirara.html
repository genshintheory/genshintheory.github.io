Combat

Kirara's E creates a dendro shield. Her Q does a dendro nuke that creates smaller bomblets like Aloy's skill. These bomblets can be clustered and pulled via anemo vacuum abilities like Sucrose E.

Using hold E will generate the shield and activate the neko state, turning Kirara into a boxcat that can move around fast, jump, and even climb up walls. Her movement speed, jumping, climbing speed, and stamina consumption while climbing are all increased. In the neko state, when Kirara runs into enemies she will deal dendro dmg (once per .5s). This neko state lasts up to 10s.

The shield lasts 12s. The press cd is 8s (so the shield can have 100% uptime, unlike baizhu). By holding E the cd can be longer up to 12s. The longer she is in the neko state, the longer the cd. Note the cd starts after the neko state ends (so she can run into the Childe dilemma).

Moreover, using the E again while the shield is active will stack the shield and make it absorb more damage up to a certain max limit. With a passive, while in the neko state, for each enemy hit, she gains a stack of 'reinforced packaging' (up to 3). When the neko state ends, each stack will increase the shield dmg absorption of her E.

All the game needs now is a shield hp bar that shows how much damage a shield absorbs like a character's hp bar. After all, Star Rail has a shield hp bar, so Genshin definitely has the capacity and talent to create and program one.

Using E will apply the dendro element to Kirara herself. What this can do is actually remove elemental debuffs on characters (from ley lines, spiral abyss, etc.). For example, if Kirara is affected with electro, then using E will apply dendro and cause quicken on Kirara. Other abilities in the game that apply elements to the character include Beidou's E and Xinyan's E. These are characters that can be used to remove elemental debuffs.

Kirara is better than baizhu because her shield does not cost 80 energy, absorbs much more damage, and can have 100% uptime.
